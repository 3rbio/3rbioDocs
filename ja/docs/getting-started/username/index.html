ユーザー名ガイドライン

3rb.ioのユーザー名ガイドラインは変更や修正ができないルールです。ユーザー名を作成する際の基本条件となります。

ガイドライン​

備考

新しいユーザー名で使用できる文字：

ゲームの見た目や体験をさまざまな設定でカスタマイズできます。以下のオプションが使用可能です：

  • ラテン文字 (a–z)
  • 数字 (0–9)
  • ウェアラブルの表示
  • 一部の特殊文字：
    1. アンダースコア ( _ )
    2. ピリオド ( . )
備考

新しいユーザー名の制限：

  • 文字数は2〜20文字であること。

  • 小文字の英字 (a–z)、数字 (0–9)、アンダースコア (_) およびピリオド (.) のみ使用可能。

  • (_) と (.) 以外の特殊文字は使用不可。

  • 連続するピリオドは使用不可 — 例：a..b は無効、.a.b や a.b は有効。

  • 大文字小文字を区別せず、自動的に小文字に変換されます。

  • 3rb.io内で一意であること — 同じユーザー名を共有できません。

  • ヘイト、嫌がらせ、性的に不適切な内容は禁止。

  • ユーザー名（およびニックネーム）は コミュニティガイドライン に従う必要があります。
