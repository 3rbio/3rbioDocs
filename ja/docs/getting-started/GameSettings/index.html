ゲーム設定

3rb.io のゲーム設定では、あなたのスタイルに合わせてゲーム体験をカスタマイズすることができます。コントロール設定からビジュアルテーマまで、ここに必要なすべてがあります。

ゲームプレイ​

備考

試合結果のスキップ!

各試合の終了時 — 死亡または時間切れの場合 — 「試合結果」の画面が表示されます。ここでは、ゲームセッションの詳細な統計情報が表示され、プレイ時間、消費した食べ物、最大質量、生存時間、食べた細胞、リーダーボードでの最高順位が表示されます。

備考

レンダリングオプション!

ゲームの見た目と感じをカスタマイズするために、さまざまなビジュアルオプションがあります。次のような要素をオンまたはオフにすることができます：

  • 名前の表示
  • スキンの表示
  • アクセサリの表示
  • アニメーション速度
  • 自動ズーム
  • マウスカーソル
  • 質量の表示
  • 食べ物の非表示
  • 境界線の非表示

これらの設定は、視覚的な体験をあなたの好みやパフォーマンスニーズに合わせるのに役立ちます。

備考

インターフェース（UI）オプション!

ユーザーインターフェースの要素を制御し、気を散らすものを減らしたり、パフォーマンスを改善したりできます。利用可能なオプションは次のとおりです：

  • XP バーの非表示 – 経験値バーを画面から削除します。
  • チャットの非表示 – ゲームチャットを非表示にして、インターフェースをクリーンにします。 :::

テーマ​

テーマ!

ゲームの外観をカスタマイズするために、お好みの背景スタイルを選択できます。利用可能なオプションは次のとおりです：

  • ヘクス agon – ヘクス agon 模様のスタイリッシュな背景
  • ライン – 線形グリッド模様の背景
  • NoN-Style – 3rb.io の標準背景

カラー!

ゲームのインターフェースの一般的なカラースキームを選択できます：

  • ダーク
  • ホワイト
  • ブラック :::

コントロール​

危険

コントロールのリセット – すべてのコントロールをデフォルト設定に戻します。

備考

より良い体験のために、ゲーム内でコントロールをカスタマイズし、リセットすることができます。利用可能なアクションは次のとおりです：

  • 餌やり – 質量を排出するためのキーを割り当てます。
  • 分割 – 単一の分割を実行します。
  • 二重分割 – すばやく 2 回分割します。
  • 三重分割 – 連続して 3 回分割します。
  • x16 分割 – 16 細胞に完全に分割します。
  • 凍結（線形分割） – 線形分割を正確に実行するために一時的に凍結します。
  • 垂直（線形分割） – 垂直な線形分割を実行します。
  • UI の非表示（クリップ用） – クリアな録画やスクリーンショットのために UI をすべて非表示にします。 :::

地域​

現在、ゲームはヨーロッパにある単一のサーバーで実行されています。将来的には、グローバルパフォーマンスの向上を目的に、追加のリージョンが追加される予定です。 :::
