Normas de Nombres de Usuario

Las normas de nombres de usuario en 3rb.io son reglas que no se pueden cambiar ni modificar. Son un requisito básico para crear tu nombre de usuario.

Normas​

info

Caracteres permitidos para nuevos nombres de usuario:

Personaliza cómo se ve y se siente el juego con varias opciones visuales. Puedes usar opciones como:

  • Letras latinas (a–z)
  • Números (0–9)
  • Mostrar accesorios (Wearables)
  • Algunos caracteres especiales:
    1. Guion bajo ( _ )
    2. Punto ( . )
info

Restricciones para nuevos nombres de usuario:

  • Deben tener entre 2 y 20 caracteres.

  • Solo se permiten letras minúsculas (a–z), dígitos (0–9), guion bajo (_) y punto (.).

  • No se pueden usar otros caracteres especiales además de (_) y (.).

  • No se permiten puntos consecutivos — por ejemplo, a..b es inválido, pero .a.b o a.b está bien.

  • No sensible a mayúsculas/minúsculas; se fuerza a minúsculas.

  • Deben ser únicos en 3rb.io — no pueden existir dos usuarios con el mismo nombre.

  • No se permite contenido de odio, acoso o sexual explícito.

  • Los nombres de usuario (y apodos) deben cumplir con nuestras Normas de la Comunidad.
